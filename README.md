# 開発言語
- Ruby 3.0.1
- Rails 6.0.4.8
# 実行手順
    $ git@github.com:yoshihiro-kawabata/TimeCook.git
    $ cd TimeCook
    $ bundle install
    $ rails db:create && rails db:migrate
    $ rails s
